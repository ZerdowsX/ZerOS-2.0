# ZerOS-2.0 / ZerDOS GUI Prototype

Минимальная ОС на **Assembly + C** с простейшей графической оболочкой в VGA text mode.

## Что теперь умеет

- загрузка через Multiboot (NASM);
- ядро на C;
- красный фон «рабочего стола»;
- синяя тестовая текстура (квадрат);
- рисуется курсор (`X`), который можно двигать;
- управление курсором через `W/A/S/D`;
- `Q` возвращает курсор в исходную позицию.

## Сборка

Требования:
- `nasm`
- `gcc` с поддержкой `-m32`
- `ld`
- `grub-mkrescue` **или** `grub2-mkrescue` (для ISO)
- `xorriso` (обычно нужен для `grub-mkrescue`)
- (опционально) `qemu-system-i386`

```bash
make
```

## Сборка ISO

```bash
make iso
```

## Запуск

```bash
make run
# или
make run-iso
```
